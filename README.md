# 🎮 2 Player Tic-Tac-Toe Game

A beginner-friendly Tic-Tac-Toe game built using **HTML**, **CSS**, and **JavaScript**. 

## Features

- Interactive 3×3 game board
- Two-player gameplay (X & O)
- Winner detection
- Draw detection
- Reset Game button
- New Game button

 ## Tech Stack

- HTML5
- CSS3
- JavaScript
